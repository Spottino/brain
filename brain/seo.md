# 🔍 SEO — Paolo AI Brain

## CORE PRINCIPLE

La SEO NON serve per portare traffico generico.
Serve per intercettare persone con un bisogno reale.

Obiettivo:
visibilità mirata → conversione

---

## SEARCH INTENT

Ogni contenuto deve intercettare uno di questi intenti:

1. Funzionale
(es. “icone app fintech”)

2. Stilistico
(es. “hand drawn icons”)

3. Commerciale
(es. “logo lettering custom”)

---

## PRIMARY KEYWORDS

Usare sempre keyword specifiche, non generiche.

Categorie principali:

### LOGHI
- custom lettering logo
- hand drawn logo
- calligraphy logo
- logo design branding

### ICONE UI
- UI icon set
- custom app icons
- fintech icons
- dashboard icons

### ILLUSTRAZIONE
- vector illustration
- hand drawn vector
- flat illustration system

---

## LONG-TAIL STRATEGY

Preferire:
- “custom fintech icon set for mobile app”
invece di:
- “icons”

Obiettivo:
meno traffico → più qualità

---

## TAG STRATEGY (marketplace)

Regola:

- Prime 5 tag = fondamentali
- Devono essere:
  - specifiche
  - coerenti
  - commerciali

Struttura:

1. funzione (logo, icon, illustration)
2. stile (hand drawn, lettering)
3. settore (fintech, food, ecc)
4. uso (app, branding)
5. caratteristica (custom, unique)

---

## TITLE STRATEGY

Formato:

[funzione] + [stile] + [uso] + [contesto]

Esempio:
Custom Hand Drawn Lettering Logo for Food Brand

---

## DESCRIPTION STRATEGY

Struttura:

1. cosa è
2. per chi è
3. cosa risolve
4. dove si usa

NON:
- descrizioni generiche
- keyword stuffing

---

## VISUAL SEO

Le immagini devono:

- comunicare subito il contesto
- mostrare utilizzo reale
- evitare mockup generici

---

## DIFFERENTIATION RULE

Ogni contenuto deve rispondere a:

Perché questo è diverso da uno stock?

Se non è chiaro → contenuto debole

---

## PLATFORM ADAPTATION

### Marketplace (Adobe Stock, ecc)
- keyword chiare
- descrizione semplice
- focus su ricerca

### Social
- meno keyword
- più concetto
- più posizionamento

---

## ANTI-SEO

Evitare:

- keyword troppo generiche
- tag casuali
- contenuti non coerenti con ricerca reale

---

## SUCCESS METRIC

- download
- impression qualificate
- conversione
